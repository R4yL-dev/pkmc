# PKMC

PKMC is a Go application for managing items, extensions, blocks, and languages with SQLite and GORM.

## Features
- Manage items, extensions, blocks, languages
- Automatic migration and seeding

## Quick Start
```sh
go mod tidy
go run cmd/pkmc/main.go
```

## Requirements
- Go 1.25+
- SQLite
- GORM
